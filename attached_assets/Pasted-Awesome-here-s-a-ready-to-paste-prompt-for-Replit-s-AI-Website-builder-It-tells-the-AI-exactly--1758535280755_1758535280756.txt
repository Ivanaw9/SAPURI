# Prompt for Replit AI Website

Build a modern, professional website for **SAPURI GROUP (PT. Sumber Alam Putera Lestari)** that **rebuilds** and **improves** the structure/content from **[https://sapuri.co.id/](https://sapuri.co.id/)** while using a clean, responsive, interactive UI.

## Tech & Architecture

* Use **React + Vite** with **Tailwind CSS** (or vanilla HTML/CSS/JS if required by the generator).
* Mobile-first, responsive up to 1440px+, semantic HTML, WCAG AA contrast.
* Keep content primarily in **Bahasa Indonesia**; prepare for optional **EN** toggle (scaffold, no need to translate all copy).
* Organize components clearly (components/, pages/, data/).

## Brand & Visual System

* **Extract brand palette from the source site/logo** and use it consistently.

  * If extraction fails, use this fallback palette:

    * `--color-primary: #0C1C8F` (deep royal blue)
    * `--color-primary-600: #1426B0`
    * `--color-accent: #FFB000` (warm accent for CTAs/hover states)
    * `--color-bg: #F6F8FB`
    * `--color-text: #101828`
    * `--color-muted: #667085`
* Rounded corners (xl/2xl), soft shadows, generous whitespace.
* Use **system fonts** or Inter.

## Pages & IA (keep URLs readable)

   * Hero with headline, short value prop, CTA buttons (“Lihat Produk”, “Hubungi Kami”).

6. **Ceramic Division**

* **Sticky header**, mega-menu hover/focus, mobile drawer menu.
* **Hero**: subtle background animation (CSS only).

* **Accordions** for specs/FAQs (keyboard accessible).
* **Carousel** for featured items (no external heavy libs).
